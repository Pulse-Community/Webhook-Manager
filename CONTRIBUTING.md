# Contributing Guidelines

Vielen Dank, dass du zu Webhook Manager beitragen möchtest!

## Wie du helfen kannst

1. Forke das Repository und erstelle einen neuen Branch für deine Änderungen.
2. Führe deine Anpassungen durch und erstelle aussagekräftige Commits.
3. Öffne einen Pull Request und beschreibe deine Änderungen kurz.
4. Stelle sicher, dass dein Code unseren Stilrichtlinien folgt und alle Tests bestehen.

Wir freuen uns über jeden Beitrag!
